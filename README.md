# QuriosNow — Coming Soon

Static Coming Soon page for QuriosNow (QuriosNow.in).

How to view

1. Open [index.html](index.html) in your browser (double-click or use `Live Server`).

Files

- `index.html` — main page
- `css/styles.css` — styles
- `js/script.js` — countdown + subscription logic

Notes

- The subscription form saves emails to `localStorage` (no server integration). Replace with your API endpoint when ready.
- Edit `LAUNCH_DATE` in `js/script.js` to change the countdown target.
